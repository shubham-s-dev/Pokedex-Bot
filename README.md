#  Pokedex Telegram Bot

This is a Python automation tool. You give it a Pokemon name, and it fetches Data (Weight, Type, Image) from the PokeAPI and sends it to your Telegram.

##  Tech Stack
* **Language:** Python
* **Libraries:** `requests`
* **API:** PokeAPI (No Key Required)

##  How to Run
1. Install Library: `pip install requests`
2. Enter your Telegram Bot Token in the script.
3. Run `python pokedex.py`
